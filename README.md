# Django-WebApp

This is a basic website built in django

## To Clone 
1. Clone:
    ```bash
    git clone https://github.com/papu-hacker/Django-WebApp.git
    cd Django-WebApp
    ```

2. Make VE:
    ```bash
    python -m venv .env
    source ./.env/bin/activate
    ```

3. Install Depende... :
    ```bash
    pip install -r requirements.txt
    ```

4. To Run:
    ```bash
    python manage.py runserver
    ```

## Then open url in browser http://localhost:8000/
