# Django-WebApp

This is a basic website built in django

## To Clone 
1. Clone:
    ```bash
    git clone https://github.com/papu-hacker/Django-WebApp.git
    cd Django-WebApp
    ```

2. Make VE:
    ```bash
    python -m venv .env
    source ./.env/bin/activate
    ```

3. To run :
    ```bash
    pip install -r requirements.txt
    ```
